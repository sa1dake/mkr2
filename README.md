# mkr2: Демонстрація Singleton та Observer

Це навчальний проєкт, розроблений для демонстрації та практичного застосування двох ключових шаблонів проєктування:

1.  **Породжувальний шаблон:** **Singleton (Одинак)**
2.  **Поведінковий шаблон:** **Observer (Спостерігач)**

## Структура Проєкту

| Клас/Інтерфейс | Роль у Шаблонах | Опис |
| :--- | :--- | :--- |
| **`EventManager`** | Singleton / Subject | Центральний менеджер, який керує підписками та розсилає сповіщення. |
| **`Observer`** | Observer Interface | Інтерфейс для всіх модулів, які можуть реагувати на події. |
| **`AuthModule`** | Concrete Observer | Обробляє події авторизації та може сам їх генерувати. |
| **`LogModule`** | Concrete Observer | Записує всі системні події у лог. |
| **`UserInterface`**| Concrete Observer | Відображає повідомлення користувачеві. |
| **`Main`** | Client | Точка входу, що ініціалізує систему та демонструє генерацію подій. |
